环境
JDK8 https://www.oracle.com/java/technologies/javase-jdk8-downloads.html 如果下的慢我晚点上传一份到QQ群
IDEA官网下载地址 https://www.jetbrains.com/idea/download/#section=windows
IDEA破解 http://idea.lanyus.com/
MySQL下载5.7版的（不要下载8的），网上找找怎么下载和配置都有
Navicat加破解版 http://www.zdfans.com/html/43925.html（MySQL是数据库，Navicat是操作数据库的可视化软件）

maven（依赖管理）的下载与安装：https://www.cnblogs.com/qxcxy-silence/p/10808321.html
Tomcat的下载与安装： https://tomcat.apache.org/download-90.cgi
https://blog.csdn.net/little__superman/article/details/88724359

基础：
知道一些基本概念，比如jar包，maven，mybatis，springboot，去了解下一些HTTP状态码对应的含义，比如200,404,500是什么，了解下MVC

前端 HTML + CSS + JS
文档的话，菜鸟教程的就挺不错的，大致过一下，知道它是怎么回事就行了，也可以去看一些视频

后台的话，Java其实跟C差不多，菜鸟教程也有Java的一些东西，可以去大致过一下（也可以自己去找视频大致过一下）
框架的话，SpringBoot+Mybatis就可以了

看这些东西最重要是知道它是用来干嘛的，具体怎么用，JavaWEB使用注解开发会比较多，一些东西不理解的网上搜一下都会有的，包括运行程序可能会报错，也可以先把错误复制上去看看是啥问题

SpringBoot的两个视频，重点是看看基础的一些东西比如什么是SpringBoot，yml配置等等，还有MyBatis的使用（MyBatis是ORM框架，差不多就是用来操作数据库的）（这些教程可能没有讲Spring的一些基本的点，比如 @RequestMapping，EL表达式什么的，需要自己去搜博客看看它用来干什么，怎么用）

SpringBoot的整个流程

（看完视频再来看）

一般是在浏览器输入URL请求，请求通过 @RequestMapping 等注解到controller层对应的方法，它就调用service层对应的方法对请求和请求附带的一些参数进行处理，service层调用dao层(mapper)来对数据库进行操作，把前端的信息保存进数据库，或者返回数据库的东西给service，再通过service返回给controller，controller就把这些数据展示给前端显示
